# RGB Color Game

## Description
A fun and interactive **RGB Color Guessing Game** built using **HTML**, **CSS**, and **JavaScript**.  
The player tries to match the displayed RGB color code by selecting the correct color from multiple options. Correct guesses increase the score, while wrong guesses reset it.

---

## Features
- Displays a random **RGB color code**  
- 6 color options to choose from  
- **Score tracking** stored in localStorage  
- Responsive grid layout for color options  
- Interactive hover effect on options  

---

## How It Works
1. A random RGB color code is generated and displayed.  
2. Six color boxes are shown, one of which matches the RGB code.  
3. Player clicks a color box to make a guess.  
   - If correct, **score increments by 1**.  
   - If incorrect, **score resets to 0**.  
4. The game continues with a new RGB color after each guess.  
5. The score is stored in **localStorage** to persist across sessions.

## How to Play
1. Open **index.html** in a web browser.  
2. Check the RGB color code displayed.  
3. Click on the color box that matches the code.  
4. Watch your **score** increase with correct guesses.  
5. Continue guessing with new colors generated automatically.
